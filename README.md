# Vue Account Management

A small account management system built with Vue 3, Vite, TypeScript, Pinia, Vue Router, Axios, and Tailwind CSS.

## Features

- Login authentication
- Token storage
- Protected routes
- Account list
- Create account
- Edit account
- Delete account
- Search accounts
- Loading and error handling
- Responsive layout

## Tech Stack

- Vue 3
- Vite
- TypeScript
- Pinia
- Vue Router
- Axios
- Tailwind CSS
